== Description ==

This plug-in for Woocommerce enables an auto address correction to be implemented in the checkout process using the Google API.

= Features =

* Address suggestions refer to the selected country.
* Compatible with all versions of WooCommerce.
* Save your customers time and avoid typing errors.
* The city and zip code for the selected address will be filled in automatically.

**Important**

A Google API key is required to complete the addresses!
