== Description ==

Dieser Plugin für Woocommerce, erlaubt es mittels Google API eine Autoadresskorrektur im Kassenvorgang zu realisieren.

= Features =

* Adressvorschläge beziehen sich auf das ausgewählte Land.
* Kompatibel mit allen Versionen von WooCommerce.
* Sparen Sie Ihrem Kunden Zeit und vermeiden Sie Tippfehler.
* Stadt und Postleitzahl für die ausgewählte Adresse werden automatisch ausgefüllt.

**Wichtig** 

Es wird ein Google API-Schüssel benötigt zum Vervollständigen der Adressen!
